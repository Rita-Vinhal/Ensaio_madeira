Ensaio à compressão em corpos de prova de madeira.
  
Foram submetidos 19 corpos de prova de diferentes espécies nativas, de aproximadamente 2x2x3cm à compressão, 3 de cada espécie.
Obtidos a tensão máxima e deformação. 
Os dados foram considerados no estágio plástico, despresados os dados iniciais de acomodação e os dados do estágio elástico.
